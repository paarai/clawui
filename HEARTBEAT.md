# Heartbeat Tasks

Run these checks each time you receive a heartbeat:

1. Read TODO.md and execute any tasks that are safe to perform without user input.
2. If any task is completed, update TODO.md to mark it done.
3. Review recent memory files and update MEMORY.md with important insights.
4. Check git status for uncommitted changes; note in daily memory if any.
5. After tasks are done, reply with HEARTBEAT_OK.
