### Self-as-Backend Browser Automation Test (13:37-13:53)
- **任务**: 用龙虾自身作为 AI 后端，自适应控制浏览器注册 Google 邮箱
- **流程**:
  1. ✅ CDP 导航到 accounts.google.com/signup
  2. ✅ 读取表单字段（firstName, lastName）
  3. ✅ 用 IIFE 包装 JS 填写 name → 值成功写入
  4. ✅ 点击"下一步" → 成功跳转到 birthdaygender 页面
  5. ⚠️ 填写 year/day 成功，但月份选择器和性别 radio 是自定义组件，CDP 直接设值无效
  6. ❌ 停滞在 birthdaygender 页面

- **关键发现**:
  - CDP `Runtime.evaluate` 不能直接用 `return`，必须用 IIFE `(function(){...})()`
  - Google 注册表单使用自定义 UI 组件（非标准 input/radio），需要坐标点击或 `Input.dispatchMouseEvent`
  - `cdp_type` 对标准 `<input>` 有效，对自定义组件无效
  - 需要添加 `cdp_click_coords`（viewport 坐标点击）和 `Target.activateTarget`（多标签切换）
